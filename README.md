# RotaPort Tek Klasor Yukleme Paketi

Bu paket, GitHub'a tum dosyalari ayni seviyede yuklemek isteyen kullanim icin hazirlandi.

## Nasil yukleyeceksin

1. GitHub reposunun kokunde yalnizca bu klasorun icindeki dosyalar olsun.
2. `rotaport-tek-klasor` klasorunun kendisini degil, icindeki dosyalari secip yukle.
3. Vercel'de:
   - `Framework Preset`: `Other`
   - `Root Directory`: bos
   - `Build Command`: bos
   - `Output Directory`: bos
